# Purpose

Set up a development environment for Matomo quick & easy.

# Usage

```
docker-compose up
```

Then open http://localhost:3000 in your web browser. Enter the following DB setup:

1. Server: mariadb
1. User: root
1. Password: pass

Follow the instructions.
